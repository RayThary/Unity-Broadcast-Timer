# 사용 방법

## 1. 프로그램 실행

프로그램을 실행하면 처음 화면에서 일반 모드와 연동 모드를 선택할 수 있습니다.

- 일반 모드: 외부 연동 없이 타이머 사용
- 연동 모드: Toonation Alertbox URL을 입력해 룰렛 결과와 타이머 연동

## 2. 일반 모드 사용

1. 처음 화면에서 일반 모드를 선택합니다.
2. TimerPanel로 이동합니다.
3. 타이머 시간을 직접 추가하거나 차감하며 사용할 수 있습니다.

Save Timer가 켜져 있으면 처음 화면으로 돌아갔다가 다시 일반 모드로 들어와도 이전 시간이 유지됩니다.  
Save Timer가 꺼져 있으면 처음 화면으로 돌아갈 때 타이머 시간이 초기화됩니다.

## 3. 연동 모드 사용

1. 처음 화면에서 연동 모드를 선택합니다.
2. Toonation Alertbox URL을 입력합니다.
3. 필요한 경우 연동 상세 설정을 변경합니다.
4. Connect 버튼을 눌러 연결합니다.
5. 연결에 성공하면 TimerPanel로 이동합니다.
6. 설정한 룰렛 결과가 수신되면 타이머 시간이 자동으로 추가되거나 차감됩니다.

## 4. Toonation Alertbox URL 입력

투네이션 후원 페이지 주소가 아니라, OBS 브라우저 소스에 사용하는 알림창(Alertbox) URL을 입력해야 합니다.

Save URL이 켜져 있고 이전에 저장된 URL이 있다면, 다음 실행 시 저장된 URL을 다시 불러올 수 있습니다.

## 5. 연동 상세 설정

연동 상세 설정에서는 룰렛 제목, Delay, Save URL을 설정할 수 있습니다.

### 룰렛 제목

타이머에 반영할 투네이션 룰렛 제목을 입력합니다.

수신한 알림 메시지의 앞부분이 설정한 룰렛 제목과 일치할 때만 타이머에 반영됩니다.

```text
노방종 룰렛 - 5분 추가
```

설정한 룰렛 제목이 `노방종 룰렛`이면 위 결과는 타이머에 반영됩니다.

```text
다른 룰렛 - 5분 추가
```

설정한 룰렛 제목과 다르면 위 결과는 무시됩니다.

여러 개의 룰렛 제목을 사용할 경우 쉼표로 구분합니다.

```text
노방종 룰렛, 방송연장 룰렛
```

룰렛 제목을 비워두면 “노방종 룰렛”이 기본값으로 적용됩니다.

### Delay

룰렛 결과를 받은 뒤 타이머에 적용하기까지 기다릴 시간을 초 단위로 입력합니다.

- 비워두면 즉시 적용됩니다.
- 숫자를 입력하면 해당 초만큼 기다린 뒤 적용됩니다.

예시:

```text
30
```

룰렛 결과 수신 후 30초 뒤 타이머에 반영됩니다.

### Save URL

입력한 Toonation Alertbox URL을 저장할지 설정합니다.

- 켜짐: 입력한 URL 저장
- 꺼짐: URL을 저장하지 않으며, 기존에 저장된 URL도 삭제

## 6. 타이머 설정

TimerSettingPanel에서는 Save Timer를 설정할 수 있습니다.

### Save Timer

현재 남은 시간을 저장할지 설정합니다.

- 켜짐: 현재 남은 시간 저장
- 꺼짐: 남은 시간을 저장하지 않으며, 기존에 저장된 시간도 삭제

Save Timer를 꺼도 현재 실행 중인 타이머 시간이 바로 초기화되지는 않습니다.  
처음 화면으로 돌아갈 때 저장 여부에 따라 시간이 유지되거나 초기화됩니다.

## 7. 타이머 조작

TimerPanel에서 타이머를 직접 조작할 수 있습니다.

- + 버튼: 입력한 분만큼 시간 추가
- - 버튼: 입력한 분만큼 시간 차감
- Apply 버튼: 입력한 값을 적용
- Close 버튼: 입력창 닫기
- Stop / Resume 버튼: 타이머 정지 또는 재개

## 8. 룰렛 결과 적용 방식

룰렛 결과에 `추가`가 포함되어 있으면 해당 시간만큼 타이머에 추가됩니다.

```text
노방종 룰렛 - 5분 추가
```

룰렛 결과에 `차감`이 포함되어 있으면 해당 시간만큼 타이머에서 차감됩니다.

```text
노방종 룰렛 - 10분 차감
```

`광`, `꽝`처럼 시간 변화가 없는 결과는 무시됩니다.

```text
노방종 룰렛 - 광
```
