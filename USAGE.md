1. 프로그램 실행

2. Toonation Alertbox URL 입력
- 투네이션 후원 페이지 주소가 아니라 OBS 브라우저 소스에 사용하는 알림창(Alertbox) URL을 입력합니다.

3. Delay 입력
- 비워두면 룰렛 결과가 즉시 적용됩니다.
- 숫자를 입력하면 해당 초만큼 기다린 뒤 타이머에 적용됩니다.
- 예: 30 입력 시 룰렛 결과 수신 후 30초 뒤 적용

4. Save URL
- 체크하면 입력한 URL을 저장합니다.
- 다음 실행 시 저장된 URL을 다시 불러오며 자동으로 연결됩니다.
- 체크하지 않으면 URL을 저장하지 않으며, 저장된 URL도 삭제됩니다.

5. Save Time
- 체크하면 현재 남은 시간을 저장합니다.
- 다음 실행 시 저장된 남은 시간을 불러옵니다.
- 체크하지 않으면 남은 시간을 저장하지 않으며, 저장된 시간도 삭제됩니다.
- 체크 해제해도 현재 실행 중인 타이머 시간은 바로 초기화되지 않습니다.

6. Connect 클릭
- 투네이션 알림창 연결을 시작합니다.
- 연결에 성공하면 설정창이 닫힙니다.

7. 타이머 조작
- + 버튼: 입력한 분만큼 시간 추가
- - 버튼: 입력한 분만큼 시간 차감
- Apply 버튼: 입력한 값을 적용
- Close 버튼: 입력창 닫기
- Stop / Resume 버튼: 타이머 정지 또는 재개
