# Unity-Broadcast-Timer

Unity 기반 투네이션 룰렛 연동 타이머 프로젝트입니다.  
Toonation Alertbox 알림 데이터를 수신하고, 룰렛 결과에 따라 타이머 시간을 자동으로 추가하거나 차감할 수 있도록 구현했습니다.

외부 연동 없이 사용할 수 있는 일반 타이머 모드도 함께 제공하여, 투네이션 연결이 필요 없는 상황에서도 타이머를 독립적으로 사용할 수 있습니다.

## 프로젝트 개요

이 프로젝트는 방송 진행 중 투네이션 룰렛 결과를 타이머에 자동으로 반영하기 위해 제작한 Unity 개인 프로젝트입니다.

Toonation Alertbox URL을 입력하면 알림 데이터를 수신하고, 지정한 룰렛 제목과 결과값을 분석하여 타이머 시간을 변경합니다.

단순 타이머 기능뿐만 아니라, 외부 알림 데이터 수신, WebSocket 연결, 룰렛 결과 파싱, 설정 저장, 화면 흐름 분리까지 직접 구현하는 것을 목표로 했습니다.

## 주요 기능

### 투네이션 룰렛 연동

- Toonation Alertbox URL 입력
- Alertbox 페이지에서 연결에 필요한 payload 추출
- WebSocket을 통한 실시간 알림 데이터 수신
- 지정한 룰렛 제목의 결과만 타이머에 반영
- 룰렛 결과에 따라 시간 추가 / 차감 처리

### 타이머 기능

- 시간 추가
- 시간 차감
- 입력값 적용
- 타이머 정지 / 재개
- 타이머 초기화
- 남은 시간 저장 옵션 지원

### 모드 분리

- 일반 모드
  - 외부 연동 없이 타이머 화면으로 바로 진입
  - 직접 타이머 조작 가능

- 연동 모드
  - Toonation Alertbox URL 입력 후 연결
  - 연결 성공 시 타이머 화면으로 진입
  - 룰렛 결과에 따라 타이머 자동 제어

### 설정 저장

- Save URL
  - 입력한 Toonation Alertbox URL 저장 여부 설정
  - 저장된 URL이 있을 경우 다음 실행 시 자동 불러오기

- Save Timer
  - 현재 남은 시간 저장 여부 설정
  - 일반 모드 재진입 시 이전 시간 유지 가능

- Delay
  - 룰렛 결과 수신 후 타이머에 반영하기까지의 대기 시간 설정

- 룰렛 제목 설정
  - 타이머에 반영할 룰렛 제목 지정
  - 여러 룰렛 제목 입력 가능
