# Unity-Broadcast-Timer

Unity 기반 투네이션 룰렛 연동 타이머 프로젝트입니다.  
Toonation Alertbox URL을 통해 알림 데이터를 수신하고, 특정 룰렛 결과에 따라 타이머 시간을 자동으로 추가하거나 차감할 수 있도록 구현했습니다.

## 프로젝트 개요

이 프로젝트는 방송 진행 중 사용할 수 있는 타이머 시스템을 Unity로 제작한 개인 프로젝트입니다.

단순한 카운트다운 타이머 기능뿐만 아니라, 투네이션 Alertbox URL을 기반으로 외부 알림 데이터를 수신하고, 특정 룰렛 결과에 따라 타이머 시간을 추가하거나 차감하는 기능을 구현했습니다.

일반 모드와 연동 모드를 분리하여 사용자가 상황에 따라 원하는 방식으로 타이머를 사용할 수 있도록 구성했습니다.

## 주요 기능

- 일반 타이머 모드
  - 외부 연동 없이 타이머를 직접 실행
  - 시간 추가 / 차감 / 적용 / 정지 / 재개 기능 지원

- 외부 룰렛 연동 모드
  - Toonation Alertbox URL 입력을 통한 연결
  - WebSocket 기반 알림 데이터 수신
  - 지정한 룰렛 제목과 결과값을 기준으로 타이머 시간 자동 반영

- 룰렛 결과 처리
  - “추가” 결과 수신 시 타이머 시간 증가
  - “차감” 결과 수신 시 타이머 시간 감소
  - “광”, “꽝” 등 시간 변화가 없는 결과는 무시

- 설정 저장 기능
  - Save URL 옵션을 통한 Alertbox URL 저장 여부 제어
  - Save Timer 옵션을 통한 남은 시간 저장 여부 제어
  - Delay 값을 통해 룰렛 결과 반영 시점 조절

- 화면 흐름 분리
  - StartPanel에서 일반 모드 / 연동 모드 선택
  - SetupPanel에서 URL 입력 및 연결
  - DetailSettingPanel에서 연동 관련 세부 설정
  - TimerSettingPanel에서 타이머 저장 관련 설정 관리

## 실행 흐름

```text
프로그램 실행
→ StartPanel
→ 일반 모드 선택
→ TimerPanel 진입

또는

프로그램 실행
→ StartPanel
→ 연동 모드 선택
→ SetupPanel에서 Alertbox URL 입력
→ 연결 성공
→ TimerPanel 진입
